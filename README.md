## 📱 Responsive Design

### Breakpoints:
- **Desktop (1024px and above)** - Full horizontal layout
- **Tablet (768px - 1024px)** - Adapted layout with hamburger menu
- **Mobile (576px and below)** - Optimized single-column layout with:
  - Centered footer content
  - Stacked navigation menu
  - Full-width form fields
  - Mobile-friendly typography

### Mobile Features:
✅ Hamburger menu icon (visible at ≤1024px)
✅ Smooth menu animations
✅ Centered footer on mobile (≤576px)
✅ Touch-friendly buttons and links
✅ Optimized image sizes

## 🚀 Getting Started

### View the Website:
1. Open `index.html` in your web browser
2. Navigate through sections using the menu
3. Explore services and gallery
4. Fill contact form to reach out

### Customize for Your Business:
1. Replace logo image in `images/waffle logo.png`
2. Update restaurant name and details in HTML
3. Replace images in the `images/` and `More images/` directories
4. Modify business hours and contact details in the Contact section
5. Update social media links in the footer
6. Adjust color scheme by modifying CSS variables:
   - `--primary-color: #3b141c` (dark brown)
   - `--secondary-color: #f3961c` (orange)
